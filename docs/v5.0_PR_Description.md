## ✨ Phase 3 新增功能

### 鱼游动（CSS offset-path）

- 8 条预定义贝塞尔曲线覆盖鱼缸全区域（水平、垂直、波浪、对角）
- 鱼随机选路径循环播放，遇边界自动重新选
- 朝向切换：`transform: scaleX(-1)` CSS 瞬间翻转
- 性能降级：≤20 条完整游动 / 21-50 条 50% 游动 / >50 条全部降级上下浮动
- Safari 15- 降级：`@supports (offset-path)` 检测，不支持的浏览器用 `float-up-down` keyframes

### 喂食动画（JavaScript 状态机）

- 状态机：IDLE → FEEDING → DETECTING → SWIMMING_TO_FOOD → EATING → DONE
- 食物颗粒 CSS keyframes 下落动画
- 多鱼同步：React Context + 单一 dispatcher，每条鱼独立判断
- 鱼嘴张合 2-3 次关键帧动画
- 进食后身体微沉 + 气泡上浮效果

### 温度天气联动（Open-Meteo API）

- 免费 API 无需申请 key（替代架构设计的"和风 API"）
- 城市级缓存：Prisma `weather_cache` 表，TTL=30 分钟，坐标四舍五入 2 位小数提高命中率
- 用户偏好：`user_preferences` 表 + `PUT /api/preferences` + `GET /api/preferences`
- 前端："我的 → 偏好 → 城市" 搜索 + 快速选择
- 优雅降级：未设城市隐藏天气块 / API 失败 fallback 默认值

### 天气背景（4 套 CSS）

- `.tank-bg--sunny`：阳光径向渐变 + 波纹
- `.tank-bg--cloudy`：灰色调 + 平静
- `.tank-bg--rainy`：深色调 + 雨滴动画
- `.tank-bg--snowy`：浅蓝 + 雪花飘落
- 切换时长 200ms CSS transition
- 鱼缸组件订阅 `weatherCode` 自动切换 className

## ⚠️ Breaking Changes

1. **Prisma schema 新增 2 个模型**：`UserPreference` + `WeatherCache`，需 `prisma migrate deploy`
2. **新增 API 端点**：
   - `GET /api/weather/:lat/:lon` — 获取天气（公开）
   - `GET /api/weather/city?city=<name>` — 按城市名获取天气（公开）
   - `PUT /api/preferences` — 更新用户偏好（需认证）
   - `GET /api/preferences` — 获取用户偏好（需认证）
3. **前端新增页面**：`/[locale]/profile` 偏好设置页（含城市选择器）
4. **鱼缸页重构**：新 `TankStage` 组件替换旧版鱼缸，鱼从静态浮动 → 贝塞尔曲线巡游
5. **i18n 新增 key**：`profile.*`, `weather.*`, `feeding.*`
