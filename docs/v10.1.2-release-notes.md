## Item 2 & 3 N/A 判定理由

- v10.1.1 的 3 个 revert commit 仅回退了 v10.1.0 的 3 个增量改动
- v10.1.0 的增量改动是: visualVariant / BUG-V12-1&2 / BUG-V12-3&4&5
- v9.1 addendum (nickname flow, rename, my-fish list, city selector, temperature) 和 v10.0 (per-tank city, weather) 不在 revert 范围内
- 因此 nickname flow (Item 2) 和 city selector (Item 3) 在 main 上始终存在，无需重新实现
